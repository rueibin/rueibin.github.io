2021-08-18
網頁程式APP
Java

前言

使用Java Swing來開發一個圖形界面的員工管理系統。 涉及Swing裡常用的元件與監廳器，包括JFrame、JPanel、JTable、JLabel、JButton與各種Layout等。 本專案採用分層結構設計，有dao層，entity層，util層等。 使用原生的JDBC技術來連接MySQL。

系統說明

    系統名稱 : Java Swing CRUD
    開發年分 : 2021年
    開發工具 : Eclipse
    開發語言 : Java 1.8
    資料庫 : MySQL 5.7
    運行環境 : Windows
    Source Code : Simple-Java-Swing-CRUD

系統架構

  1. dao層：資料存取類
  2. entity層：封裝物件業務類與資料表對應
  3. exception層：例外類
  4. test層：測試類
  5. util層：工具類，如資料庫連線類
  6. view層：所有的視窗類

使用技術

  1. JDBC
  2. Java Swing
  3. Singleton（資料庫連線）

功能說明

  1. 程式啟動顯示主界面，顯示所有員工資料
  2. 點擊查詢按鈕，能夠根據姓名模糊查詢，顯示符合查詢條件的員工資料
  3. 新增員工信息，點擊新增按鈕，將員工資料儲存到資料庫中，保存成功，給出提示添加成功。
  4. 選中某個員工，點擊修改按鈕，將員工資料儲存到資料庫中，修改成功，給出提示修改成功。
  5. 選中某個員工，點擊刪除按鈕，提示是否刪除
