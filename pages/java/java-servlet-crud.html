簡單的Java Servlet CRUD應用

2021-08-27
網頁程式APP
Java

前言

使用Java Servlet來開發一個員工管理系統。包含Servelet、JSP、Filter、GSON、DBUtil與C3P0等。 抽取出一個BaseServet減少系統Servlet數量，以及獨立出DAO層與Service層， 使用單例模式來連接MySQL，與DBUtil簡化JDBC操作，採異步操作。

系統說明

    系統名稱 : Java Servlet CRUD
    開發年分 : 2021年
    開發工具 : Spring Tool Suite 3.9.7.RELEASE
    開發語言 : Java 1.8
    資料庫 : MySQL 5.7
    運行環境 : Windows Tomcat
    Source Code : Simple-Java-Servlet-CRUD

系統架構

  1. dao層：資料存取類
  2. entity層：封裝物件業務類與資料表對應
  3. service層：業務邏輯類
  4. servlet層：轉發與接收的動作
  5. test層：測試類
  6. util層：工具類，如資料庫連線類

使用技術

  1. Servlet 3.0 & JSP
  2. C3P0
  3. DBUtil
  4. jQuery
  5. Ajax

功能說明

  1. 資料列表，分頁顯示。
  2. 新增員工資料與編輯員工，以及檢查資料格式，條件如下：
    1. employee name：
      1. 檢查姓名必填；
      2. 檢查員工名稱可以使用；
      3. 檢查員工名稱可以是2-5中文或是6-16英文與數字的組合；
    2. email：
      1. email必填；
      2. email格式不正確；
  3. 單一刪除與批次刪除，點擊刪除按鈕，提示是否刪除。
