簡單的Laravel 7 CRUD應用

2021-09-06
PHP
Laravel

使用Laravel來快速開發一個員工管理系統，前端使用blade模版，後端使用Model，Controller，router，validate與paginate等技術

系統說明

    系統名稱 : Laravel 7 CRUD
    開發年分 : 2020年
    開發工具 : PhpStorm
    開發語言 : PHP 7.2.5+
    資料庫 : MySQL 5.7
    運行環境 : Windows
    Source Code : Simple-Laravel-7-Ajax-CRUD

使用技術：

  1. Ajax
  2. blade模版

功能說明：

  1. 資料列表，分頁顯示。
  2. 新增員工資料與編輯員工，以及檢查資料格式，條件如下：
    1. employee name：檢查姓名必填。
    2. email：
      1. email必填；
      2. email格式不正確；
  3. 單一刪除與批次刪除，點擊刪除按鈕，提示是否刪除。

資料庫：

員工資料表

欄位名稱 欄位說明 型態 NULL 其它
id 員工id INT(11) NOT NULL AUTO_INCREMENT,PK
name 員工姓名 VARCHAR(10) NOT NULL
gender 性別 INT(11) NOT NULL
email email VARCHAR(10) NOT NULL
dept_id 部門id INT(11) NOT NULL

部門資料表

欄位名稱 欄位說明 型態 NULL 其它
id 部門id INT(11) NOT NULL AUTO_INCREMENT,PK
name 部門名稱 VARCHAR(10) NOT NULL
