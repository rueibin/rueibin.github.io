簡單的Vue2.x CRUD應用

2021-11-10
Web
Vue

前言

使用Vue2.x來開發一個員工管理系統。使用腳手架建立專案，採前後分離的方式，前端使用Vue，後端使用Springboot。

系統說明

    系統名稱 : Vue2.x CRUD
    開發年分 : 2021年
    開發工具 : WebStorm 2020.1.1
    開發語言 : Vue
    資料庫 : MySQL 5.7
    運行環境 : Windows
    Source Code : Simple-Vue2-CRUD（此專案只有前端）

系統架構

  1. assets資料夾：存放靜態資源
  2. components資料夾：存放所有的元件
  3. router資料夾：設定路由
  4. utils資料夾：存放自定義的工具

使用技術

  1. Vue 2.x
  2. Axios
  3. Boostrap 3.x

功能說明

  1. 程式啟動顯示主界面，顯示所有員工資料
  2. 新增員工信息，點擊新增按鈕，將員工資料儲存到資料庫中。
  3. 選中某個員工，點擊修改按鈕，將員工資料儲存到資料庫中。
  4. 選中某個員工，點擊刪除按鈕，提示是否刪除
