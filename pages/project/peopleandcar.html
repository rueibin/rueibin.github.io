早期第一個PHP專案，人車管制系統

2021-08-18
作品
PHP

前言

這個專案是我任職網管期間，使用原生PHP開發的小系統，包含前後端，也是我的第一個PHP專案，值得紀念。

那個時候公司有改善提案的制度，資訊隸屬於管理部底下，每個月每個部門都要提2件改善提案， 其實如果站在資訊的角度要提改善提案，是提不完的，所提的案件都可以為公司解決問題。

某一天，辦公室傳來，xx主管在公司嗎?xx保管在品保室嗎?小白公務車有人開嗎?小藍貨車還在公司嗎? 腦袋突然傳來一個畫面，那就來寫一個可以查詢人員與車輛的系統好了。

每天花一點時間來coding，心裡OS，終於可以寫程式了，哈哈...。

系統主要是為了即時管控人員以及車輛，透過開啟網頁的方式，瀏覽已外出的人員及公用車輛，其中包含油費補助，會計可以透過個人出差費查詢補助出差人員費用。

系統說明

    系統名稱 : 人車管理系統
    開發年分 : 2012年
    開發語言 : PHP
    資料庫 : MySQL
    運行環境 : Centos Apache

使用技術

  1. Html&Css
  2. jQuery
  3. PHP

功能說明

  1. 前台：
    1. 外出人員與車輛的瀏覽功能。
  2. 後台：
    1. 部門管理：可以新增、查詢、修改、刪除功能公司部門。
    2. 使用者管理：新增人員與設定部門。
    3. 管理者管理：管理者依權限等級來決定使用的功能。
    4. 車輛管理：新增公務車與員工車輛。
    5. 油費管理：每月新增補助油費，油費有異常可以修改。
    6. 外出人車管理：由警衛記錄外出人員外出事由和使用交通工具。
    7. 個人出差費瀏覽：可以查詢每月個人的出差明細與油費補助。
    8. 個人外出查詢：可以查詢外出記錄。
